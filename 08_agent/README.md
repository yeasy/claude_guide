# 第八章：Agent 架构设计

AI Agent 是能够自主完成复杂任务的系统。本章介绍如何基于 Claude 构建 Agent。

---

## 什么是 AI Agent？

Agent = LLM + 工具 + 记忆 + 规划

一个 Agent 能够：
- 理解复杂任务
- 制定执行计划
- 调用工具完成子任务
- 根据结果调整策略
