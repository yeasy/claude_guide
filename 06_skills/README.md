# 第六章 Skills 技能系统

Skills 是 Claude 2025 年推出的新能力，让你可以创建可复用的定制化工作流。

---

## 什么是 Skills？

Skills 是包含指令、脚本和资源的文件夹，Claude 可以自动发现并应用它们。可以把 Skills 想象成为 Claude 定制的“技能包”。

---

## 本章学习目标

- [ ] 理解 Skills 的概念和结构
- [ ] 使用内置 Skills 处理常见任务
- [ ] 创建自定义 Skills
- [ ] 组合多个 Skills 实现复杂工作流
