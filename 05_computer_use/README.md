# 第五章：Computer Use 计算机操控

Computer Use 是 Claude 最具突破性的能力，让 AI 能够像人类一样操作计算机界面。

---

## 革命性意义

传统 AI 只能处理文本和数据。Computer Use 让 Claude 能够：

- 直接操作桌面应用程序
- 填写表单、点击按钮
- 在没有 API 的系统中工作
- 执行端到端的自动化任务
