# 第七章 Agentic Coding 与 Claude Code

Claude 已成为开发者的强力编程助手。本章介绍如何充分利用 Claude 的编程能力。

---

## Agentic Coding 是什么？

Agentic Coding 指 Claude 作为自主编程代理，能够：

- 理解代码库结构
- 读写文件
- 运行测试
- 自动修复问题
- 管理 Git 提交
